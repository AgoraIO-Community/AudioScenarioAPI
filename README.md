# Audio Scenario API

### 概述

为降低开发者的集成难度，声网为 1v1 连麦和秀场直播连麦场景提供了音频 Scenario 场景化 API。你只需要调用一个 API 即可实现 1v1 连麦和秀场直播连麦场景下的音频最佳实践。
